## 추가 학습
### builder() method
그냥 생성자를 메서드로 쓰는거
```java
// 생성자 사용하면 이렇게 쓰는데
Order order = new Order(boormiId, dreamiId, 5000, "MATCHING", "PACKAGE");

// builder 메서드 쓰면 이렇게. 단 Order에 lombok의 @Builder 어노테이션 있어야 함
Order order = Order.builder()
                   .boormiId(boormiId)
                   .dreamiId(dreamiId)
                   .deliveryAmount(5000L)
                   .itemCd(ItemCode.PACKAGE)
                   .build();
```
